# ShopGalaxy

This is a responsive *Amazon-style e-commerce website clone* built using *HTML and CSS*.  
It includes a navigation bar, hero section, product categories, and a footer.  

## Features
- Responsive navbar with logo, search bar, account, orders, and cart  
- Product sections with different categories displayed in a grid layout  
- Hero banner with call-to-action link  
- Footer with multiple columns and links  
- Fully responsive design for desktop, tablet, and mobile  
- Font Awesome icons for location, search, cart, and menu  

## Project Files
- index.html, style.css, images (logo, hero, product boxes)  

## How to View
Open index.html in any browser to see the project live.
